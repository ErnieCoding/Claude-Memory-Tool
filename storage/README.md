# Storage Directory

Эта директория содержит файлы для работы приложения.

## Структура

### `user_files/`
- Загруженные пользователем файлы
- Claude может **только читать** эти файлы
- Файлы загружаются через веб-интерфейс или копируются вручную
- Поддерживаемые форматы: JSON, TXT, XML, PDF, CSV, XLSX, XLS

### `responses/`
- Ответы, сгенерированные Claude
- Claude может **создавать, редактировать и удалять** файлы в этой директории
- Здесь хранятся результаты анализа, отчёты и другие выходные данные

## Использование

**Для загрузки файлов:**
1. Через веб-интерфейс: http://localhost
2. Вручную: скопируйте файлы в `storage/user_files/`

**Для просмотра результатов:**
- Откройте директорию `storage/responses/` в проводнике
- Или просмотрите через веб-интерфейс в разделе "Сохранённые ответы"

## Важно

Файлы в этих директориях **не добавляются в Git** (см. `.gitignore`).
При первом запуске директории будут пустыми.
